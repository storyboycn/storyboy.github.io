生而为人、我很抱歉

发表于 2019-05-01 | 分类于 日记
| 字数统计: 1.2k | 阅读时长 ≈ 3


伸出手臂，伸向天空，紧紧地搂住那颗小星星。伸出手臂，踮起脚尖，让我们够得到蓝色的天空。

松子最根本的伤害是来自于父亲。因为妹妹病重，父亲总是偏爱妹妹多一点。扮鬼脸也是为了博得父亲的开心和关注。可在父亲眼中，他们失重的天平永远不会平衡。松子便把这些痛苦归于妹妹。。。哪怕在自己回家后，看到死去父亲的日记，也没有原谅妹妹。。。父亲的不公平对待，在松子的心中种下了缺爱的种子。

松子被学校辞退，明明自己不是小偷，却被当作小偷被辞退。后来那个作家却是因为太爱松子，留下一句【生而为人，我很抱歉】而去自杀。松子和作家生活的日子里，其实并不幸福。作家让松子去做浴室女郎赚钱，并且对她实施家暴。这样生活下的松子并没有反抗或者抱怨，因为她知道作家是爱她的。世上有这一个爱她的人，她才足以有希望地活下去。

作家死后，松子做了作家死对头的情妇。可以这样来说，松子的世界又有了爱，生活又继续充满了希望。可这希望又在松子得知，这个男人根本不爱她，和她在一起也是因为这自卑感而破灭。

回过一趟家的松子决定只要不是一个人，只要离家远一点，和一个叫小野寺的男人做起了交易。在小野寺背叛自己之后，她杀了小野寺，失去了希望。决定要自杀的松子遇到了能给自己带来希望的理发师。进监狱之后，也正是因为抱着这份爱，松子在浑浑噩噩的生活中挺到了出狱的那天。希望还是一如既往地破灭了。在她满心欢喜地站在理发店前，看到理发师搂着自己的妻子，抱着自己儿子的时候，留下了眼泪。

在美容院遇到泽村惠的松子尝到了友情的甜蜜。可这终究是友情。。。友情再多，朋友再多，也终归于，她是一个男人的另一半。

之后松子的情感生活就只出现一人，那就是以前的混混学生龙洋一。他们相爱了。之后的生活中，龙洋一惹上了事，松子为了去找他，成了瘸子。龙洋一在监狱里忏悔，松子在监狱外等待。龙洋一带着自己会再次伤害松子的害怕，离开了松子。

松子在自己的弟弟告诉自己以后不要回来的时候，她开始将自己封锁，不在乎别人对自己的看法，看着河流发呆过日子。

在医院，她遇到了泽村惠，躺在家里，看着天花板上像妹妹屋子里的千纸鹤，给在天堂的妹妹剪了一个短发，看到了生活的希望，在河边寻找泽村惠名片的时候，告诉孩子们早点回家，她想重新融入这个生活，妹妹接她走上了天堂的路。

和松子一起生活过的人都知道，松子喜欢唱歌，而且唱的很好听。那位自杀的作家说，生而为人，我很抱歉。其实，人生在这世，就是个矛盾体，向外求取爱和物资。人们总是希望自己能够做一个不缺爱，不缺物资的人。在我看来这是不可能的，因为你生而为人，爱不可或缺。

如果一些痛苦又充满爱的经历摆在人的面前，让人选择是希望自己从未遇见，还是再从头来过。答案应该是从头来过吧。因为生而为人，与众不同的就是有感受和经历爱的能力。我想松子亦是如此。

希望每个女生都能够明白，没有得到父母足够多的爱，不要把所有希望寄托在一个男人身上，而忘却生活，忘却所有其他爱你的人。把自己没有得到过的爱，传递给世上另外一个没有得到这份爱的人。要相信，自己什么样子，自己的生活就是什么样子。愿天堂的松子能够抱着希望和幸福，快乐地活下去。也愿生活中的松子，抱着生而为人，我很幸运的态度去面对生活，面对蓝色的天空。
